03 · 광합성
1엽록체의 구조와 기능
구조특징 및 기능
막 구조외막 + 내막의 2중막으로 둘러싸여 있다.
틸라코이드납작한 주머니 모양의 막 구조. 여러 개가 쌓여 그라나를 이룬다. → 막에 광합성 색소·전자 전달계·ATP 합성 효소 존재
스트로마틸라코이드를 제외한 나머지 공간. 광합성에 관여하는 여러 효소가 모여 있다.
핵심엽록체는 내부에 복잡한 막 구조(틸라코이드)가 발달하여 표면적을 넓혀 광합성 효율을 높인다.
2광합성 색소
틸라코이드 막에 있는 광합성 색소에서 빛에너지가 흡수된다.
엽록소 a초록색. 광합성에서 가장 중심적인 역할을 하는 색소 (반응 중심)
엽록소 b초록색. 보조 색소
카로티노이드황색·적황색(카로틴·잔토필). 빛에너지를 흡수해 엽록소에 전달하는 보조 색소
③빛 흡수율과 광합성 속도
  • 엽록소는 청자색·적색 빛을 주로 흡수 → 이 영역에서 광합성이 가장 활발 → 광합성에 필요한 에너지가 주로 엽록소에서 흡수됨을 의미
  • 카로티노이드계 색소는 초록색 빛을 흡수하여 엽록소에 전달
흡수 vs 작용흡수 스펙트럼(파장별 빛 흡수율)과 작용 스펙트럼(파장별 광합성 속도)이 거의 일치 → 엽록소가 광합성에 필요한 빛에너지를 주로 흡수함. 단, 엽록소가 거의 흡수하지 않는 파장에서도 광합성이 일어나는 이유는 보조 색소(카로티노이드)가 그 빛을 흡수하기 때문.

3광합성 과정 — 전체 개요
물이 분해되어 산소 발생 + 이산화 탄소가 포도당으로 환원. 필요한 에너지는 빛에너지에서 공급.

4명반응
명반응 — 엽록체의 틸라코이드 막에서 빛에너지를 흡수하여 ATP와 NADPH를 생성하는 과정 (광계 + 전자 전달계 작용)
① 물의 광분해빛에너지로 물 분해 → 수소 이온(H⁺)·전자·산소 발생. 전자는 전자 전달계를 거쳐 NADP⁺로 전달 → NADPH 생성
② 광인산화빛에너지를 이용한 ATP 합성 과정
③ 광계반응 중심(엽록소 a + 1차 전자 수용체) + 안테나 색소로 구성. 안테나 색소가 빛에너지를 모아 반응 중심 엽록소 a로 전달 → 엽록소 a가 고에너지 전자 방출 후 산화

⑤광인산화 (화학 삼투)
  • 빛에너지 흡수로 형성된 H⁺의 농도 기울기를 이용하여 ATP를 합성
  • 스트로마의 H⁺을 틸라코이드 안(내부)으로 운반 → H⁺ 농도 기울기 형성 → 기울기에 따라 H⁺이 스트로마 쪽으로 확산 → ATP 합성 효소에서 ATP 합성

3DNA의 구조
①DNA의 구성
구성 단위뉴클레오타이드 — 인산 : 당 : 염기 = 1 : 1 : 1로 결합
당디옥시리보스
염기아데닌(A)·구아닌(G)·사이토신(C)·타이민(T)
염기 계열고리 구조해당 염기
퓨린 계열고리 구조 2개A, G
피리미딘 계열고리 구조 1개C, T
폴리뉴클레오타이드 — 뉴클레오타이드끼리 결합하여 형성. 당-인산 골격을 따라 5′ → 3′ 방향성을 나타낸다.
②DNA의 입체 구조
  • 폴리뉴클레오타이드 두 가닥이 나선 모양으로 꼬인 이중 나선 구조
  • 두 가닥의 염기가 쌍을 이룰 때 A은 항상 T과, G는 항상 C와 상보적으로 결합 (수소 결합)
  • 샤가프 법칙 → DNA 이중 나선에서 A 수 = T 수, G 수 = C 수
수치DNA 이중 나선: 염기쌍 10개당 3.4 nm(한 바퀴), 염기쌍 사이 간격 0.34 nm, 지름 2 nm. 두 가닥은 역평행(서로 반대 방향)으로 배열.

1유전자 재조합 기술
유전자 재조합 기술 — 서로 다른 생물의 DNA를 인위적으로 자르고 연결하여 새로운 유전자 조합의 DNA를 만드는 기술
기본 원리특정 유전자를 DNA 운반체에 삽입 → 재조합 DNA 제작 → 세포에 넣어 발현시킴
이용 예인슐린·인터페론·혈전 용해제·간염 백신 등 의약품 대량 생산, 형질 전환 농작물·가축 개발

2핵치환
핵치환 — 어떤 세포에서 핵을 꺼내어 핵을 제거한 다른 세포에 이식하는 기술
  • 이용 예 : 복제 동물의 생산, 체세포 복제 배아 줄기세포 생산
  • 복제 동물 과정 : 체세포에서 핵 추출 → 핵을 제거한 난자에 핵치환 → 세포 분열로 배아 형성 → 대리모 자궁에 배아 이식 → 복제 동물 탄생
3조직 배양
조직 배양 — 생물의 조직 일부나 세포를 떼어 내어 영양분이 든 인공 배지에서 증식시키는 기술
  • 이용 예 : 복제 식물의 생산, DNA 재조합·핵치환 기술로 만들어진 세포·조직을 배양
  • 복제 식물 과정(당근) : 세포를 영양 배지에서 배양 → 캘러스 형성 → 분리해 영양 용액에서 재배양 → 캘러스로부터 형성된 배를 배지로 옮겨 배양 → 어린 식물 → 다 자란 식물
4세포 융합
세포 융합 — 서로 다른 두 종류의 세포를 융합하여 새로운 잡종 세포를 만드는 기술
  • 이용 예 : 잡종 식물의 생산, 단일 클론 항체의 생산
  • 잡종 식물 과정(포마토) : 토마토·감자의 세포벽 제거 → 원형질체 → 세포 융합 → 핵 융합 → 잡종 세포 → 조직 배양 → 포마토
정리4대 생명 공학 기술 — 유전자 재조합(DNA 자르고 연결), 핵치환(핵 이식), 조직 배양(인공 배지 증식), 세포 융합(두 세포 융합). 조직 배양과 세포 융합은 서로 연계되어 사용된다.

▣단일 클론 항체
단일 클론 항체 — B 림프구와 암세포를 세포 융합하여 만든 잡종 세포로부터 얻은 한 종류의 항체
  • 잡종 세포는 항체 생산 + 인공 배지에서 분열 가능 + 수명 반영구적
  • 이용 예 : 병원균 감염 여부 진단, 암 치료, 임신 진단 키트
▣유전자 치료
유전자 치료 — 유전적 결함이 있는 사람에게 정상 유전자를 넣어 이상 유전자를 대체하거나 정상 단백질이 합성되게 하여 질병을 치료하는 방법
  • 이용 예 : 암·유전병 같은 난치병 치료
  • 과정(중증 복합 면역 결핍증) : 정상 유전자를 바이러스(DNA 운반체)에 삽입 → 환자의 골수 세포에 정상 유전자 도입 → 골수 세포를 환자의 골수에 이식

1줄기세포
줄기세포 — 몸을 구성하는 근육 세포·신경 세포·혈구 등으로 분화할 수 있는 능력을 가진 미분화 세포
이용 예 : 관절염 환자에 주사하여 연골 세포 재생, 치매·파킨슨병 환자에 주사하여 신경 세포 재생
종류획득 방법특징 및 문제점
배아 줄기세포초기 배아에서 핵치환 기술로 획득몸을 구성하는 어떤 세포로든 분화 가능 / 생명 윤리 문제 발생
성체 줄기세포탯줄 혈액이나 성체의 골수 등에서 획득생명 윤리 문제 피할 수 있음 / 증식 어렵고 모든 세포·조직으로 분화 불가
유도 만능
줄기세포		성체의 체세포를 역분화시켜 획득생명 윤리 문제 자유로움 + 면역 거부 반응 없음 / 역분화 과정에서 유전자 변이 가능
2유전자 변형 생물체(LMO)의 개발과 이용
유전자 변형 생물체 (LMO) — 생명 공학 기술을 이용하여 만들어진 새로운 조합의 유전 물질을 가진 생물
식량 자원병충해에 강한 옥수수, 제초제 저항성 콩, 가뭄·염분에 강한 벼, 우유 많이 생산하는 젖소, 슈퍼 연어, 프로비타민 A 생산 황금쌀, 항산화 물질 많은 보라색 토마토
의약품 생산인슐린·생장 호르몬 생산 세균, 혈액 응고 단백질을 젖으로 분비하는 염소
환경 정화중금속(카드뮴·납) 흡수 식물, 기름·독성 유기 화학물 분해 세균
신재생 에너지바이오 에탄올용 고구마, 세포벽이 쉽게 분해되는 작물, 당→에탄올 효소 대량 생산 미생물

3서술형[4점, 부분 점수]

4서술형[4점, 부분 점수]
광합성은 명반응과 탄소 고정 반응으로 구분된다. 두 반응이 각각 일어나는 장소를 쓰고, 명반응에서 생성되어 탄소 고정 반응에 사용되는 두 가지 물질을 쓰시오.
5옳은 것 선택형[6점, 각 1점]
광합성의 전자 흐름에 대한 설명이다. 괄호 안에서 옳은 것을 고르시오.
조 　 건
  • 비순환적 전자 흐름에는 광계 Ⅰ과 광계 Ⅱ가 ( 모두 관여한다 / 관여하지 않는다 ).
  • 비순환적 전자 흐름에서는 ATP가 ( 생성된다 / 생성되지 않는다 )와/과 NADPH가 ( 생성된다 / 생성되지 않는다 ).
  • 순환적 전자 흐름에는 광계 ( Ⅰ / Ⅱ )만 관여한다.
  • 순환적 전자 흐름에서는 ( ATP / NADPH )만 생성된다.
  • 물의 광분해 결과 수소 이온, 전자와 함께 ( 산소 / 이산화 탄소 )가 발생한다.

  • 이중 나선을 이루는 두 가닥에서 G의 수와 ( C / A )의 수는 같다.
